## Cómo colaborar
1. Cada módulo tiene su carpeta y su hilo de chat.  
2. Todo código/notebooks se sube vía Pull Request; al menos un compañero revisa.  
3. Usa el tablero **Projects** para mover tus tareas de _Backlog_ → _Done_.
